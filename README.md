
# Strngth Bld Gym Floridablanca Website

A modern responsive fitness website built using HTML, CSS, and JavaScript.

## Features

• Responsive design for mobile, tablet, and desktop  
• Animated hero sections  
• Services page with training programs  
• About page with facility overview  
• Contact page with location map  
• Mobile hamburger navigation  
• Smooth hover and scroll animations  

## Technologies Used

HTML5  
CSS3  
JavaScript
